What is a DC?

DC is a domain controller which means it will gather other PC's around itself.

On a domain controller, you can only sign in using a domain user account (except if you start the DC in the “Directory Services Recovery Mode”). On other domain computers (clients or member servers), you can always sign in with a domain user account (highly recommended) or a local user account (not recommended).

What is a RODC

It is a read only Domain Controller

contains an AD in which only read operations are possible

can only replicate data from writable DCs, and never replicate data to other DCs → only inbound replication

is often installed in less secure environments (e.g. branch offices)
